# DATA MAHASISWA MENGGUNAKAN SPRING PADA JAVA
Project di atas merupakan implementasi sederhana dari aplikasi Spring Boot yang memanfaatkan konsep MVC (Model–View–Controller) dan ORM (Object Relational Mapping) untuk mengelola data mahasiswa. Melalui entitas ModelMahasiswa, aplikasi memetakan struktur tabel database ke dalam objek Java, sementara MahasiswaRepository menyediakan operasi CRUD secara otomatis tanpa perlu menulis SQL. Controller berperan sebagai pengatur alur logika dengan menyediakan menu interaktif berbasis console untuk menampilkan data, menambahkan data baru, dan mengecek koneksi ke database. Kelas utama aplikasi menginisialisasi Spring Boot dan menjalankan controller melalui mekanisme CommandLineRunner. Secara keseluruhan, project ini menunjukkan bagaimana Spring Boot mampu menyederhanakan konfigurasi aplikasi, mempercepat proses integrasi dengan database melalui JPA, serta memberikan struktur pengembangan yang rapi, modular, dan mudah diperluas.
